About App

WLAN Drop – Offline File Sharing Made Easy

WLAN Drop is a cross-platform file sharing app built using Flutter. It lets you transfer files between Android phones, tablets, TVs, and Windows PCs using your local network — either over Wi-Fi or a mobile hotspot. No internet connection is required. Just connect your devices to the same network and start sharing instantly.

Awesome Features of WLAN Drop

Works seamlessly on Android phones, tablets, TVs, and Windows PCs using your local network.

Receivers can simply scan a QR code or open a URL in their browser to download files — no app install required.

How to Use

User Friendly Interface and Very Easy to Use

Open the app and share files easily over your local network without any hassle.

To Send Files:
  • Open the app and tap Send.
  • Select the files you want to share.
  • Or, from your file manager or gallery, tap Share and choose WLANdrop.
  • A QR code and a link will appear.
  • Alternatively, use the Send button at the top-right corner to send files directly to another device running the app.
To Receive Files:
  • Scan the QR code or open the link in a browser.
  • Or, tap Receive on the receiving device to start the transfer.
  • Supported media files can be streamed instantly without downloading.
